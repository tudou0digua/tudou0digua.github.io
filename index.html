Hexo搭建Github-Pages博客填坑教程

1.安装Node.js

安装nvm

也可以用curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash

安装完成之后，重启Terminal(终端)，能识别nvm --version命令代表安装成功。

安装Node.js

$ nvm list

$ sudo npm install hexo-cli -g

$ cd test
$ npm install hexo —save

$ cd test
$ git clone https://github.com/theme-next/hexo-theme-next themes/next

title: test //博客的名字

author: name //你的名字
language: zh-CN //语言 中文

theme: next //刚刚安装的主题名称

deploy:
type: git //使用 Git 发布
repo: https://github.com/XXX/XXX.github.io.git // GitHub io仓库
branch: master //GitHub io分支

$ npm install hexo-server –save

$ npm install hexo-deployer-git –save

$ hexo g

$ hexo s

INFO Start processing
INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

$ hexo clean && hexo g && hexo d
